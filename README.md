# GlowEr

Telegram bot for GlowEr beauty studio: browse services, view portfolio and reviews, auto-confirm bookings with optional staff selection, and prompt for post-appointment photo reviews.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
